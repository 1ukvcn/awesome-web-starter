# Awesome Web Starter

一个现代化的 Vue + Vite 项目启动模板。

## 特性

- ⚡️ Vite - 极速的开发体验
- 🖖 Vue 3 - Composition API
- 🎨 开箱即用的样式配置
- 📦 开箱即用的项目结构

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
